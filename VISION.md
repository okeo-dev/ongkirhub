# Vision

OngkirHub is an open-source shipping integration framework that provides a unified interface for shipping providers.

Developers integrate OngkirHub once and can switch between shipping providers without changing their application code.

## Principles

- Provider-neutral architecture
- Consistent API contracts
- Extensible plugin system
- Self-hostable
- Open source first

## Non-Goals

- Not a courier company
- Not a shipping marketplace
- Not a fulfillment platform
