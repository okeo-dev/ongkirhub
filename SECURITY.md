# Security policy

## Supported versions

Security fixes are provided for the latest minor release on `main`.

## Reporting a vulnerability

Please report security issues privately to the maintainers of `okeo-dev/ongkirhub` rather than opening a public issue.

Include:

- A description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will acknowledge receipt and coordinate a fix before public disclosure when appropriate.
